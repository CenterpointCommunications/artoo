ARDrone

Driver

Allows user to send flight control commands to an ARDrone. Flight control commands are those used to tell the ARDrone to take off, land, and or any number of other autonomous flight manuvers.

How to establish a connection and setup the driver

Communication with the ARDrone's flight control interface takes place using a WiFi connection. The ARDrone is a WiFi access point, so it normally can be the only device you connect to, without some additional effort to reconfigure the drone itself to put it into infrastructure mode. 

require 'artoo'
connection :ardrone, :adaptor => :ardrone, :port => '192.168.1.1:5556'
device :drone, :driver => :ardrone, :connection => :ardrone

Commands:

start → drone
Starts up the drone.
stop → drone
Stops the drone.
hover → drone
Causes the drone to hover at current altitude.
land → drone
Causes the drone to begin its landing sequence.
take_off → drone
Causes the drone to take off, if already started.
emergency → drone
Causes the drone to make an immediate emergency landing.
front_camera → drone
Causes the drone to turn on the front-facing camera.
bottom_camera → drone
Causes the drone to turn on the bottom-facing camera.
up → drone
Causes the drone to increase altitude.
down → drone
Causes the drone to decrease altitude.
left → drone
Causes the drone to bank to the left.
right → drone
Causes the drone to bank to the right.
forward → drone
Causes the drone to bank forward.
backward → drone
Causes the drone to bank backward.
turn_left → drone
Causes the drone to rotate to the left.
turn_right → drone
Causes the drone to rotate to the right.
reset_watchdog → drone
Reset the watchdog in the drone.
led(color_animation, strength, duration) → drone
Turn on the drone's built in LEDs with the color_animation passed, strength and duration; color_animation might be one of the following:
  • :blink_green_red
  • :blink_green
  • :blink_red
  • :blink_orange
  • :snake_green_red
  • :fire
  • :standard
  • :red
  • :green
  • :red_snake
  • :blank
  • :right_missile
  • :left_missile
  • :double_missile
  • :front_left_green_others_red
  • :front_right_green_others_red
  • :rear_right_green_others_red
  • :rear_left_green_others_red
  • :left_green_right_red
  • :left_red_right_green
  • :blink_standard
animate(animation, duration) → drone
Animate the ardrone to do flips and turns depending on selection, available animation options are:
  • :phi_m30_deg
  • :phi_30_deg
  • :theta_m30_deg
  • :theta_30_deg
  • :theta_20_deg_yaw_200_deg
  • :theta_20_deg_yaw_m_200_deg
  • :turnaround
  • :turnaround_go_down
  • :yaw_shake
  • :yaw_dance
  • :phi_dance
  • :theta_dance
  • :vz_dance
  • :wave
  • :phi_theta_mixed
  • :double_phi_theta_mixed
  • :flip_ahead
  • :flip_behind
  • :flip_left
  • :flip_right

How to use

Example of controlling an ARDrone.

require 'artoo'

connection :ardrone, :adaptor => :ardrone, :port => '192.168.1.1:5556'
device :drone, :driver => :ardrone, :connection => :ardrone

work do
  drone.start
  drone.take_off

  after(15.seconds) { drone.hover.land }
  after(20.seconds) { drone.stop }
end

Example of ARDrone doing a flip.

require 'artoo'

connection :ardrone, :adaptor => :ardrone
device :drone, :driver => :ardrone, :connection => :ardrone

connection :navigation, :adaptor => :ardrone_navigation, :port => '192.168.1.1:5554'
device :nav, :driver => :ardrone_navigation, :connection => :navigation

work do
  on drone, :ready => :fly
  drone.start(nav)
end

def fly(*data)
  drone.take_off
  after(10.seconds) { drone.animate(:flip_ahead, 1000) }
  after(15.seconds) { drone.hover.land }
  after(20.seconds) { drone.stop }
end
