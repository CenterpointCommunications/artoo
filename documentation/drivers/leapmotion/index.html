Leap Motion

Driver

Allows user to interact with Artoo devices using a Leap Motion controller. The Leap Motion uses its built in cameras to track the user's hand motions, and translations those to events that Artoo can use to trigger actions or control hardware.

How to establish a connection and setup the driver

require 'artoo'
connection :leapmotion, :adaptor => :leapmotion, :port => '127.0.0.1:6437'
device :leapmotion, :connection => :leapmotion, :driver => :leapmotion

Events

We're working on the documentation for the Leap Motions's events. Thanks for your patience.

Example receiving hand motion events from a Leap Motion.

require 'artoo'

connection :leapmotion, :adaptor => :leapmotion, :port => '127.0.0.1:6437'
device :leapmotion, :connection => :leapmotion, :driver => :leapmotion

work do
  on leapmotion, :hand => :wave
end

def wave(sender, hand)
  puts hand.inspect
end
