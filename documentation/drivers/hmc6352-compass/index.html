HMC6352 Digital Compass

Driver

Allows user to read heading from HMC6352 Digital Compass via i2c interface

How to establish a connection and setup the driver

For Arduino:

require 'artoo'
connection :arduino, :adaptor => :firmata, :port => '/dev/ttyACM0'
device :compass, :driver => :hmc_6352_compass, :connection => :arduino, :interval => 0.5

Example using a HMC6352 Digital Compass.

require 'artoo'

connection :arduino, :adaptor => :firmata, :port => '/dev/ttyACM0'
device :compass, :driver => :hmc_6352_compass, :connection => :arduino, :interval => 0.5

work do
  on compass, :heading => proc { |caller, data| puts "heading: #<Middleman::CoreExtensions::Data::DataStore:0x00000005817400>"}
end
