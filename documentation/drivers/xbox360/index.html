XBox360 Controller

Driver

Allows user to interact with Artoo devices using a XBox360 game controller.

How to establish a connection and setup the driver

require 'artoo'
connection :joystick, :adaptor => :joystick
device :controller, :driver => :xbox360, :connection => :joystick, :interval => 0.1

Events

We're working on the documentation for XBox360 driver events. Thank you for your patience.

Example receiving button events from XBox360 controller joystick.

require 'artoo'

connection :joystick, :adaptor => :joystick
device :joystick, :driver => :xbox360, :connection => :joystick, :interval => 0.1

work do
  on joystick, :joystick => proc { |caller, value|
    puts 'joystick ' + value[:s].to_s, value[:x], value[:y]
  }
  on joystick, :button_a => proc { |*value|
    puts 'a button'
  }
end
